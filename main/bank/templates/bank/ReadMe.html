Opis Aplikacji

Aplikacja ma za zadanie symulować działanie banku. Umożliwia tworzenie kont urzytkowników oraz zakładanie kont bankowych oraz podstawowe operację transakcyjne.

Do stworzenia aplikacji został użyty Python 3.10.1 oraz środowisko programistyczne Visual Studio Code. Aplikacja jest oparta na bibliotece Django 4.0. Oraz dodatkowych bibliotek rozszerzających możliwości djnago.

  • rest_framework
  • rest_framework.authtoken
  • crispy_forms
  • crispy_bootstrap5
Aplikacja jest również backendem do aplikacji frontend'owych napisanych w React i Angular.
